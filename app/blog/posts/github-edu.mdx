GitHub na své stránce poskytuje studentům, ale i učitelům, řadu výhod a nástrojů pro výuku a vývoj softwaru.

### GitHub Education

GitHub Education je program, který nabízí studentům a učitelům řadu výhod a nástrojů pro výuku a vývoj softwaru. Mezi hlavní výhody patří možnost získat zdarma přístup k GitHub Pro, GitHub Copilot, GitHub Codespaces a dalším nástrojům. Díky tomu mohou studenti a učitelé vytvářet, sdílet a spravovat své projekty a kódy.

### Výhody GitHub Education

GitHub Education má i partnery, mezi které patří například Heroku, AWS, JetBrains, DigitalOcean.

Například JetBrains nabízí svá IDE studentům zdarma, AWS poskytuje studentům kredity na své služby a DigitalOcean nabízí studentům a učitelům slevu na své služby.

### GitHub Copilot

Osobně největší výhodou je GitHub Copilot. Copilot je AI nástroj, který doplňuje kód za vás. Stačí napsat komentář a Copilot vám doplní kód.

```python	
# Vytvoření pole s náhodnými čísly
pole = [random.randint(0, 100) for i in range(10)]
```

Jelikož web programuji v Next.js, tento odstavec pro mě vygeneroval GitHub Copilot. Zvládá tedy i přirozený jazyk. A ten kód si sám zvolil (já chtěl TypeScript 😢).

### GitHub Pro

GitHub Pro je placená verze GitHubu, která nabízí řadu výhod. 

Já používám odznáček, aby všichni viděli, že jsem PRO. Ale důležitější pro mě je, že nemusím mít veřejné repozitáře, abych je hostoval na GitHub Pages.
